#### Stack Kata

A test driven implementation of the Stack Kata wherein the developer creates a version of the abstract **stack** data structure.

<br />

#### Running Tests

Migrate from **Python** to **Java** code, all-the-while ensuring that 100% code coverage is maintained.

<br />

#### Hspec Test Framework

* When executing tests, the following is suggested:

```
$ ## suggested cli test execution
$ mvn test
```

<br />

* When looking for test coverage:

```
$ ## suggested cli test execution
$ mvn test && open target/site/jacoco/index.html
```
